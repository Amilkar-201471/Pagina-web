Arduino Electrónica

- Resumen -

En esta practica se hizo algo muy interesante, lo cual fue disear desde cero nuestro propio carro que se puede controlar por bluetooth desde le telefono.

- Introducción -

En esta práctica, se diseñó y construyó un robot móvil capaz de recoger pelotas, controlado a través de un celular mediante Bluetooth. El proyecto integró los conocimientos adquiridos durante el semestre, desde el diseño en aplicaciones como SolidWorks o Catia, hasta la creación del circuito y la programación en Arduino. El objetivo fue desarrollar un sistema automatizado, eficiente y controlado de forma remota, eliminando la necesidad de operación manual. La estructura del robot se fabricó utilizando corte láser o impresión 3D, aplicando lo aprendido en prácticas anteriores para completar el proyecto con éxito.

- Materiales -

  • Placa de Arduino UNO
  • Cable de transferencia
  • Una pila
  • Un protoboard
  • Módulo bluetooth
  • Llantas con motor
  • Controladores de motores
  • Filamento

Desarrollo

Desarrollo Para llevar a cabo la práctica, lo primero fue diseñar el robot en SolidWorks con el objetivo de analizar su comportamiento y funcionamiento. Se crearon distintas piezas que facilitaran el movimiento del vehículo y permitieran recoger las pelotas, incluyendo un espacio para colocar el Arduino y el circuito de control. Una vez finalizado el diseño, se imprimieron las partes utilizando filamento PLA. Después de obtener las piezas impresas, se ensamblaron y ajustaron con tuercas y tornillos hasta completar la estructura del robot.
