- Resumen -

Basicamente lo que hicimos en esta practica fue aprender las bases del html con dos pequeñas paginas webs, que se conectaban entre ellas, a continuación explicaremos todo con mas detalle.

- Introducción -

HTML, o Lenguaje de Marcado de Hipertexto, es el estándar utilizado para crear y estructurar contenido en la web. Permite organizar textos, imágenes, enlaces y otros elementos multimedia mediante el uso de etiquetas. HTML es fundamental para el desarrollo de sitios web, ya que define la estructura básica de una página, permitiendo a los navegadores mostrar el contenido de manera adecuada.

- Materiales -

  • Navegador web
  • Conocimientos minimos de como usar visual studio

- Resultados -

Los resultados de nuestra practica los podemos dividr en:

  • Estetica: Se debe de ver bien, debe de ser atractica a simple vista y ordenado
  • Efectividad: La pagina no debria de tener ningun tipo de problema a la hora de cargar y debe de cumplir con lo que se solicita
  • Accesibilidad: La pagina debe de ser accesible al navegar y utilizar

    - Conclusiones -

    En esta practica llegamos a la conclusion de que programar en HTML, es mas sencillo que en C+, pero aún asi tiene su grado de dificualtad, pero aparte de eso aprendimos lo necesario para poder programar en HTML.
